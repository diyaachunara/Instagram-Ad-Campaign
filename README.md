# Instagram Ad Campaign Performance Tracker

This project focuses on analyzing Instagram advertisement campaign data to evaluate overall performance and provide actionable insights for marketing optimization.  

Using metrics such as Total Impressions, Total Clicks, Average CTR, ROI, and Engagement Score, this dashboard helps in identifying high-performing campaigns, understanding audience behavior, and improving decision-making.

The dashboard is designed using an Instagram-themed color palette to reflect brand consistency and enhance visual storytelling.
